# Data Quality And Lineage Book

## Purpose

Document where metrics come from, how they are transformed, and what limitations users should understand.

## Required Fields Per Metric

- metric name
- definition
- source system
- transformation summary
- refresh cadence
- owner
- known limitations

## Staff-Facing Explanation Rule

- If a metric can differ from the source system because of timing, filters, or data cleaning, state that clearly in plain language.
