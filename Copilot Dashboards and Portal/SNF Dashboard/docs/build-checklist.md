# Build Checklist

1. Define the first user role and the first live use case.
2. Create the data-source inventory: PCC, NetHealth, Dataverse, Graph, SharePoint, OneDrive, Power BI, SQL, Postgres.
3. Decide which data becomes Bronze, Silver, and Gold.
4. Define the master entities in the structured store.
5. Create the role-based dashboard pages.
6. Create the assistant intents and button-first flows.
7. Add alert rules for falls, pressure injury, readmission, sepsis, med errors, and elopement.
8. Define the audit and access-control model.
9. Add CI/CD for agent, dashboard, and workflow changes.
10. Run Playwright smoke tests against the live UI.
11. Validate refresh targets and alert response times.
12. Validate clinical review and escalation paths with a human owner.
