16. Advance Care Planning Documentation

## Shared Conversation Pattern

- Start with role and goal.
- Ask for the smallest usable patient, unit, or timeframe context.
- Use buttons for repeated choices.
- Request free text only for exceptions or clinically meaningful nuance.
- Escalate when the answer could change care.

## Shared Tool Pattern

- Read from knowledge sources for policy and guideline grounding.
- Call workflow tools for alerts, review queues, or documentation tasks.
- Use browser automation only for live UI verification and admin workflows.
