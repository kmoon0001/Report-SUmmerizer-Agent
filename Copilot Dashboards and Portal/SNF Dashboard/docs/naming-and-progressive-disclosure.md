# Naming And Progressive Disclosure

## Naming Rules

- Use names that describe the user-facing purpose.
- Avoid `Untitled`, generic abbreviations, and file-only labels.
- Keep action titles, topic names, and knowledge titles aligned.

## Disclosure Rules

- Use buttons or closed lists for the first turn.
- Ask the highest-value choice first.
- Defer free text until the model truly needs it.
- Route ambiguous requests to a concrete menu.
