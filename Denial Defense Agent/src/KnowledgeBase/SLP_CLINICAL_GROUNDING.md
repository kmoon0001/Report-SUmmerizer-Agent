# SLP_CLINICAL_GROUNDING.md

**Metadata:**
- **Role**: Specialized SLP Clinical Logic
- **Grounding**: ASHA (American Speech-Language-Hearing Association)
- **Description**: Specialist rules for Dysphagia, Speech-Language, and Cognitive-Communication.

## 1. Dysphagia (Swallowing) Standards
- **Rule**: If a swallow study is referenced, AI must extract consistency requirements (Thickened, Pureed, etc.).
- **Platinum Check**: Reference MBSS or FEES results as evidence of medical necessity.

## 2. Cognitive-Communication
- **Rule**: Must use objective measures: SLUMS, MoCA, or BCAT.
- **Justification**: Link cog-comm deficits to functional safety (e.g. "Impulsivity increases fall risk").

## 3. ASHA NOMS Alignment
- **logic**: Pattern analysis of Functional Communication Measures (FCMs).
