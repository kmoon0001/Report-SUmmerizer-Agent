# Pacific Coast Therapy Hub (v3.0 Platinum)

## 🚁 Overview
This repository contains the **Ensign Clinical Swarm**, a unified ecosystem of specialized AI agents designed to automate the 24-hour clinical documentation lifecycle for Skilled Nursing Facilities (SNF). 

### **The Swarm Specialists**
*   **🛰️ SNF Agent Command Center**: The primary orchestrator and dispatcher for the entire fleet.
*   **🔬 Ensign Clinical Synthesis Lab**: Unified engine for longitudinal Case Histories and Progress/Discharge/Recertification summaries.
*   **🛡️ Denial Defense Agent**: Revenue protection hub using 7-layer XAI (SHAP/LIME) to audit for skilled necessity.
*   **⚖️ Ensign Regulatory Hub**: Unified V3 specialist for SimpleLTC triggers and MDS 3.0 compliance.
*   **🩺 TheraDoc**: Real-time clinical documentation assistant and senior peer-reviewer.

## 📐 Architecture
Built on **Microsoft Learn Platinum Standards**, the swarm uses a **"Hub-and-Spoke"** model:
1.  **Universal Context**: All agents share a `Global.SwarmContext` for seamless clinical state hand-offs.
2.  **EHR Integration**: Codified for **PointClickCare** and **NetHealth Optima** via Fabric.
3.  **Zero-Friction UX**: Optimized for mobile and desktop using Adaptive Cards and choice-driven logic (Minimizing text input).

## 🚀 Deployment
Deployed as a unified **Power Platform Solution.** 
*   **Trigger**: Automated 05:00 AM data-sync from PCC for daily clinical prep.
*   **Telemetery**: Unified logging to the `cr917_snf_telemetry_logs` Dataverse table.
