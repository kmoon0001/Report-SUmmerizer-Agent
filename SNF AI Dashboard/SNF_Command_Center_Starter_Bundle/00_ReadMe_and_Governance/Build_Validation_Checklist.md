# Build Validation Checklist

## Governance

- Clinical sponsor assigned
- Compliance owner assigned
- Technical owner assigned
- Steering review scheduled

## Data

- Data sources identified
- Critical identifiers mapped
- Consent and privacy fields mapped
- Data quality checks defined

## Assistant

- Instructions reviewed
- Topic map reviewed
- High-risk escalation paths defined
- Warning labels and disclaimers approved

## Workflows

- Trigger logic documented
- Human escalation owner documented
- Retry and rollback behavior documented
- EHR integration points validated

## Dashboard

- Clinical question documented for each page
- Role-based access defined
- RLS validation plan documented
- Performance targets documented

## Release Gate

- Clinical validation complete
- Compliance review complete
- Technical smoke test complete
- Audit logging validated
