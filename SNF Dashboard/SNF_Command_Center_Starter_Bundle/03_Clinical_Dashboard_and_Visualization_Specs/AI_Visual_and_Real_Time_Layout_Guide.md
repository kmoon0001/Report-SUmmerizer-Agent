# AI Visual And Real-Time Layout Guide

## Purpose

Define when to use Power BI AI visuals and when to use Fabric real-time dashboards.

## AI Visual Rules

- Use Key Influencers for exploratory driver analysis, not proof of causation.
- Use Decomposition Tree for guided breakdown of risk, quality, or operational variance.
- Use Smart Narrative for concise page summaries.
- Use Q&A only when field naming and semantic meaning are well governed.
- Use anomaly detection only on suitable time-series visuals with enough periods to interpret.

## Real-Time Layout Rules

- Reserve the most visible positions for patient safety, staffing pressure, and urgent operational signals.
- Keep live tiles simple and glanceable for nursing stations and wall displays.
- Use report pages for deeper analysis and drillthrough.
- Keep bedside and mobile layouts focused on one patient or one task at a time.
