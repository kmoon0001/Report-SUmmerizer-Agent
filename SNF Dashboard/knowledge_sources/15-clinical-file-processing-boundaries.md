# Clinical File Processing Boundaries

Use file intake only for the current operational or clinical-support task.

Rules:

- Prefer the smallest file that answers the question.
- Supported runtime file types are CSV, PDF, TXT, JPG, PNG, WebP, and nonanimated GIF.
- Keep images under 15 MB, PDFs under 40 pages, and TXT or CSV files under 180 KB.
- Use current exports, not stale downloads.
- Route policy, survey, and audit packets through governed review paths.
- Do not assume a file is safe to process if the minimum-necessary scope is unclear.
- If the file may contain extra PHI, stop and ask the user to reduce it first.
- Treat uploaded content as source material for review and routing, not automatic truth.
- Escalate ambiguous or high-risk findings to the relevant clinician, compliance owner, or privacy owner.
