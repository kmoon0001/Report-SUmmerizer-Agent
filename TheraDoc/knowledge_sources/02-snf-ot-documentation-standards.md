# SNF OT Documentation Standards

## Purpose
Reference for OT documentation in skilled nursing facilities. Use this source for ADL, IADL, UE, cognitive-perceptual, safety, and adaptive equipment documentation guidance.

## Daily Note Essentials
- ADLs, IADLs, UE function, coordination, cognition, safety, or adaptive equipment addressed
- Assist level and cueing required
- Measurable functional performance
- Response to treatment and limiting factors
- Why skilled OT was required
- Plan for the next visit

## Skilled OT Language
- Emphasize therapist instruction, adaptation, graded task demand, safety training, compensatory strategy training, and analysis of functional barriers.
- Link interventions to toileting, dressing, grooming, feeding, bathing, item retrieval, UE use, or cognitive safety in daily routines.

## Measurable OT Examples
- Completed UB dressing with Min A and verbal cues for sequencing and hemi-dressing technique.
- Performed toileting transfer with CGA using grab bar and safety cues for hand placement.
- Engaged in seated reaching and coordination tasks to improve functional use of dominant UE during grooming.

## Avoid
- Generic statements with no ADL relevance.
- Describing exercises without functional carryover.
- Missing safety rationale, cueing, or barriers.
