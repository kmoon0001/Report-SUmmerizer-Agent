# Approved Note Examples Library

## Purpose
Pattern library for high-quality skilled SNF therapy notes. Use examples for style, structure, and level of detail. Do not copy facts that are not present in the current patient encounter.

## PT Daily Example Pattern
Subjective context, functional mobility interventions, assist level and cueing, measurable performance, assessment of deficits and skilled need, next visit plan.

## OT Daily Example Pattern
ADL or UE-focused intervention, cueing and safety detail, measurable functional response, assessment of remaining barriers, next-step plan.

## ST Daily Example Pattern
Swallowing or cognitive-communication target, strategy training, cueing level, measurable task performance, safety or recommendation statement, plan.

## Audit Example Pattern
- Strengths: clear response to treatment, measurable gait distance, safety detail
- Gaps: missing skilled rationale, vague plan, no cueing level
- Suggested fix: replace vague statements with concise measurable wording
