# CMS SNF Services MLN Documentation Requirements

## Purpose
Supplemental CMS reference for SNF documentation expectations and improper-payment risk reduction.

## Use This Source For
- SNF documentation completeness
- Documentation that supports skilled therapy billing and review
- Common risk areas that can trigger denials or requests for additional documentation

## Key Guidance
- Document why therapy is skilled, not just what was done.
- Include function, response, and measurable change whenever possible.
- Match the note content to the plan of care and visit purpose.
- Avoid vague, repetitive, or unsupported statements.

## Best Used With
- Medicare Benefit Policy Manual Chapter 15
- Skilled necessity and compliance guide
- Discipline-specific SNF standards
