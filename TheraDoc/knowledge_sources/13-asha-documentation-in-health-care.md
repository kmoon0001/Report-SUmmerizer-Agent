# ASHA Documentation In Health Care

## Purpose
SLP-focused documentation reference for skilled speech-language pathology services in SNF and post-acute settings.

## Use This Source For
- ST documentation expectations
- Cognitive-communication and swallowing service justification
- Measurable response, cueing, and caregiver education language
- Safety, diet, and recommendation wording

## Key Guidance
- Show the specific communication, cognitive, or swallowing problem being addressed.
- Document cueing hierarchy, accuracy, carryover, and safety impact.
- Tie recommendations to observed performance and functional participation.
- Use precise language when documenting diet, liquid, or swallow strategies.

## Best Used With
- SNF ST documentation standards
- Skilled necessity and compliance guide
- Approved note examples library
