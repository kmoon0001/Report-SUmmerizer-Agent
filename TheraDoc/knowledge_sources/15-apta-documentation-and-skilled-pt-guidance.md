# APTA Documentation And Skilled PT Guidance

## Purpose
PT documentation reference for skilled physical therapy language, measurable mobility reporting, and treatment justification.

## Use This Source For
- PT daily, progress, evaluation, recertification, and discharge documentation
- Gait, transfer, balance, endurance, strength, and mobility language
- Assist level, device use, and fall-risk phrasing
- Skilled rationale for progression, modification, or monitoring

## Key Guidance
- Document the mobility problem, the therapist skill, and the measurable result.
- Show why the intervention required clinical judgment and not routine repetition.
- Include patient response, tolerance, and safety issues when relevant.
- Use functional outcomes rather than exercise-only descriptions.

## Best Used With
- SNF PT documentation standards
- Skilled necessity and compliance guide
- Net Health style and formatting reference
