# MAC Contractor Guidance

## Purpose
Supplemental contractor-specific guidance for jurisdiction-level interpretation of therapy documentation and billing expectations.

## Use This Source For
- Contractor-specific documentation nuances
- Jurisdiction-specific review risk
- Supplemental coverage interpretation

## Key Guidance
- Treat contractor guidance as supplemental to CMS manuals, not a replacement for them.
- Use only the guidance that matches the applicable jurisdiction or payer context.
- Prefer contractor guidance for edge cases, not as the primary source for general note generation.

## Best Used With
- CMS therapy documentation sources
- Skilled necessity and compliance guide
- Local payer policy when applicable
